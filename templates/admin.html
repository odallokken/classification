Domain → Classification Mapping

Each row maps an incoming caller domain to a classification level. When a call arrives, the policy server inspects the caller's remote_alias (e.g. sip:alice@example.com), looks up the domain (example.com) here, and applies that classification level to the meeting via the Pexip Client API. An elapsed-time conference timer is added to every meeting.

Unknown domains use the default level L{{ default_level }}.
